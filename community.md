# Get involved in Grin's community

## Keybase Chat
The main place for real time and asynch communication. End to end encrypted, works over TOR, and supports command line. [Grincoin team on keybase竊余(https://https://keybase.io/team/grincoin)

### Bi-weekly meetings
* every tuesday, at 15:00 [UTC](http://www.timebie.com/std/utc.php).
* **development** topics one week, in the **grincoin#dev** channel.
* **governance** topics the other week, in the **grincoin#general** channel.
* open for all.
* documented in the /grin-pm github repo:
   * [upcoming meetings & agendas竊余(https://github.com/mimblewimble/grin-pm/issues?q=is%3Aissue+is%3Aopen+label%3Ameetings)
   * [previous meeting notes竊余(https://github.com/mimblewimble/grin-pm#meeting-notes)

### How to join
1. Install and setup keybase.
2. Join the grin community:
   * **From the app:** Select "Teams" >> "Join a team" and then type "grincoin"
   * **From the command line:** `keybase team request-access grincoin`
